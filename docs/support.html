BizCaption サポート

提供者: Kazuma Yoshimoto

BizCaption は、英語の会議・商談をリアルタイムで文字起こし・翻訳する iOS アプリです。このページでは、よくある質問と問い合わせ方法をご案内します。

よくある質問

サブスクリプションの解約方法

iPhone の「設定」アプリ → ご自身の名前（Apple ID）→「サブスクリプション」→「BizCaption」→「サブスクリプションをキャンセル」から解約できます。解約しても、現在の請求期間が終了するまでは引き続きご利用いただけます。

無料利用枠について

初回限定で累計5分まで翻訳機能を無料でお試しいただけます。無料枠は一度きりで、アプリの再インストール等では再付与されません。

翻訳が正確でない場合

本アプリは外部の音声認識・翻訳サービスを利用したリアルタイム翻訳を提供していますが、翻訳の完全な正確性は保証できません。重要な商談・契約に関する内容は、必ず原文や相手方と直接確認してください。

履歴の削除

翻訳履歴はお使いの端末内に保存されています。アプリ内の履歴画面から、個別または一括で削除できます。音声保存をオンにした場合の音声ファイルも端末内に保存され、履歴削除時に削除されます。

音声保存とシャドーイング

設定で「音声を端末内に保存」をオンにしたセッションだけ、会話音声を端末内に保存できます。保存した音声は履歴から再生でき、文字起こし・翻訳とあわせてシャドーイング練習に利用できます。音声ファイルは BizCaption のクラウドには保存されません。

マイクの権限について

翻訳機能を利用するにはマイクへのアクセス許可が必要です。iPhone の「設定」→「BizCaption」→「マイク」からオン/オフを切り替えられます。

お問い合わせ

上記で解決しない場合や、アプリの不具合・ご要望がある場合は、App Store の BizCaption ページから「デベロッパWebサイト」経由でお問い合わせいただくか、GitHub の Issues でご連絡ください。
