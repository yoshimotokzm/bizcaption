BizCaption サポート

最終更新日: 2026-05-11  |  提供者: Kazuma Yoshimoto

BizCaption は、Microsoft Azure AI Speech / Microsoft Translator（Microsoft Corporation）を利用したリアルタイム音声翻訳アプリです。このページでは、よくある質問と問い合わせ方法をご案内します。

よくある質問

音声はどこに送信されますか？

翻訳機能を利用すると、翻訳セッション中のマイク音声と、音声から生成される文字起こしテキストが Microsoft Azure AI Speech / Microsoft Translator（Microsoft Corporation）に送信されます。送信目的は、リアルタイム文字起こし、字幕表示、翻訳です。

クラウド処理に同意しない場合でも使えますか？

履歴閲覧、履歴削除、設定、購入管理などは利用できますが、新規翻訳セッションは開始できません。クラウド処理への同意は、アプリ内の設定から撤回または再同意できます。

BizCaption のサーバーに音声や字幕は保存されますか？

BizCaption のサーバーは、音声本文、文字起こし本文、翻訳本文を永続保存しません。翻訳履歴と、ユーザーが有効にした場合のローカル音声ファイルは、ユーザーの端末内に保存されます。

会議アプリで使うには何が必要ですか？

現在の審査ビルドでは、端末のマイク入力を使って字幕と翻訳テキストを表示します。Zoom、Microsoft Teams、Google Meet 等の会議アプリと併用する場合は、会議音声が端末マイクで拾える状態か事前に確認してください。

リモート参加者の音声も翻訳できますか？

端末マイクで取得できる音声であれば翻訳できます。ヘッドホン利用時など、端末マイクが会議音声を拾えない環境では認識されない場合があります。

会議参加者への通知は必要ですか？

会議、通話、商談、講演等で本アプリを使う場合、音声がリアルタイム翻訳のためクラウド処理されることを、必要に応じて参加者へ通知してください。録音、保存、議事録化、共有を行う場合は、適用される法令や所属組織の規則に従い、必要な同意を得てください。

翻訳が正確でない場合

本アプリはリアルタイム翻訳を補助するものであり、翻訳の完全な正確性は保証できません。契約、価格、納期、医療、法律、金融、安全に関わる重要事項は、必ず原文、相手方、公式文書、専門家など信頼できる手段で確認してください。

履歴や保存済み音声の削除

翻訳履歴はアプリ内の履歴画面から削除できます。ローカル音声保存を有効にした場合の保存済み音声ファイルも、アプリ内から削除できます。

サブスクリプションの解約方法

iPhone、iPad、Mac の App Store アカウント設定からサブスクリプションを管理・解約できます。アプリ内の設定画面からもサブスクリプション管理画面への導線を提供します。

購入の復元

アプリ内の設定画面から購入を復元できます。購入状態は Apple StoreKit / App Store の情報に基づいて確認されます。

マイクの権限について

翻訳機能を利用するにはマイクへのアクセス許可が必要です。OS の設定アプリから BizCaption のマイク権限をオン/オフできます。

お問い合わせ

不具合、ご要望、データ取扱いに関する相談は、GitHub Issues からご連絡ください。公開される可能性があるため、機密情報、個人情報、会議内容、音声データ、API キー、購入情報の全文を投稿しないでください。
