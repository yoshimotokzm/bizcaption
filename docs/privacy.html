BizCaption プライバシーポリシー

最終更新日: 2026-05-11  |  提供者: Kazuma Yoshimoto

重要: BizCaption は、リアルタイム文字起こし、字幕表示、翻訳のため、翻訳セッション中のマイク音声と、音声から生成される文字起こしテキストを Microsoft Azure AI Speech / Microsoft Translator（Microsoft Corporation） に送信します。送信前にアプリ内で、送信されるデータ、送信先、利用目的を表示し、ユーザーの明示的な同意を取得します。

1. はじめに

Kazuma Yoshimoto（以下「当社」）は、リアルタイム音声翻訳アプリ「BizCaption」（以下「本アプリ」）を提供します。

本プライバシーポリシーは、本アプリにおけるデータの取得、利用、送信、共有、保存、削除、同意撤回、ユーザーの選択について説明します。

2. 本アプリの基本方針

  • 翻訳処理には Microsoft Azure AI Speech / Microsoft Translator（Microsoft Corporation）を利用します。
  • 当社サーバーは、Microsoft Azure AI Speech の秘密情報をアプリ内に保存しないため、短命の認証情報を発行します。
  • 当社サーバーは、音声本文、文字起こし本文、翻訳本文を永続保存しません。
  • 翻訳履歴と、ユーザーが有効にした場合のローカル音声ファイルは、ユーザーの端末内に保存されます。
  • 広告SDK、第三者広告、データブローカー共有、クロスアプリトラッキングは利用しません。
  • 収集・送信するデータは、本アプリの翻訳、利用量管理、購入確認、不正利用防止、障害対応に必要な範囲に限定します。

4. 第三者 AI サービスへの送信

本アプリは、ユーザーがアプリ内で明示的に同意した後、翻訳セッション中のマイク音声と、音声から生成される文字起こしテキストを Microsoft Azure AI Speech / Microsoft Translator（Microsoft Corporation） に送信します。送信目的は、リアルタイムの文字起こし、字幕表示、翻訳です。

送信される可能性がある情報は、音声チャンク、音声から生成される文字起こしテキスト、翻訳元/翻訳先言語、セッションID、翻訳方向、通信に必要な技術メタデータです。氏名、メールアドレス、連絡先、位置情報、広告識別子は、翻訳処理のためには送信しません。

当社は、Microsoft Azure AI Speech / Microsoft Translator 側で任意に音声を保存する設定を有効化しません。ただし、インターネット通信や外部クラウドサービスの処理は、Microsoft の契約条件、プライバシー・セキュリティ条件、法令に従って行われます。当社は、同等以上の保護を提供する第三者サービスのみを利用する方針です。

当社は、音声、文字起こし、翻訳を広告目的で第三者に提供しません。また、当社はこれらのデータをユーザープロファイル作成やクロスアプリトラッキングに利用しません。

6. 当社サーバーの役割

当社サーバーは、外部サービスの秘密情報をアプリ内に保存しないための短命認証情報の発行、利用枠管理、購入検証、不正利用防止を行います。

  • Microsoft Azure AI Speech 用の短命認証情報を発行します。
  • 短命のセッション lease を発行します。
  • StoreKit JWS と DeviceCheck token を検証します。
  • 利用量を Azure Table Storage 等で管理します。
  • 音声本文、文字起こし本文、翻訳本文、認証情報全文をログに保存しません。

7. 端末内に保存するデータ

本アプリは、翻訳履歴をユーザーの端末内に保存します。履歴には、会議・商談の内容、個人情報、営業秘密、顧客情報が含まれる場合があります。

ユーザーがローカル音声保存を有効にした場合、音声ファイルも端末内に保存されます。ローカル音声保存はいつでもオフにできます。保存済み音声ファイルと翻訳履歴は、アプリ内から削除できます。

当社は v1 では iCloud 同期を提供しません。将来 iCloud 同期やサーバー保存を提供する場合は、本ポリシーとアプリ内同意導線を更新します。

8. 利用目的

  • リアルタイム文字起こし、字幕表示、翻訳
  • 翻訳履歴、保存済み音声、復習機能の提供
  • 無料利用枠、有料プラン、追加パックの利用量管理
  • 購入状態の確認、復元、不正利用防止
  • 障害調査、セキュリティ、過剰利用検知、コスト管理
  • ユーザーサポート、法令・App Store Review・第三者サービス契約への対応

10. ユーザーの選択と権利

  • マイク権限を OS 設定から拒否または取り消すことができます。
  • Microsoft Azure AI Speech / Microsoft Translator への送信同意をアプリ設定から撤回できます。
  • 同意を撤回した場合、新規翻訳セッションは開始できません。
  • 端末内音声保存をアプリ設定からオン/オフできます。
  • 翻訳履歴、保存済み音声ファイルをアプリ内から削除できます。
  • サブスクリプションは App Store のアカウント設定から管理・解約できます。
  • 当社に問い合わせ、サーバー側記録に関する相談、削除依頼、訂正依頼を行うことができます。

11. 会議参加者への通知

会議、通話、商談、講演、ウェビナー等で本アプリを使用する場合、ユーザーは、音声がリアルタイム翻訳のためクラウド処理されることを必要に応じて参加者へ通知し、適用される法令、勤務先・取引先・会議主催者の規則、守秘義務に従って必要な同意を得てください。

録音、保存、議事録化、共有を行う場合も、ユーザーの責任で必要な許可と同意を取得してください。

12. App Store Privacy Nutrition Label との関係

App Store Connect では、実装内容に基づき、少なくとも Audio Data、User ID、Purchase History、Product Interaction を App Functionality、Tracking なしとして申告対象として扱います。Product Interaction は、利用時間、利用枠、セッション監査など、本アプリの利用量管理に必要な情報です。実装変更により収集データが変わる場合、App Store Privacy Nutrition Label と本ポリシーを更新します。

13. App Tracking Transparency

本アプリは広告目的のトラッキングを行いません。第三者広告SDK、データブローカー共有、クロスアプリ・クロスサイトのユーザー追跡を行いません。そのため、現行版では App Tracking Transparency の許可ダイアログを表示しません。

14. 子どものプライバシー

本アプリは、13歳未満の子どもを対象としていません。EEA等、より高い年齢基準が適用される地域では、その基準に従います。

15. 国際的なデータ移転

Microsoft Azure、Apple、ユーザーの所在地、サポート運用、法令上の要請により、データが日本国外を含む他の国または地域で処理される場合があります。当社は、適用される法令および第三者サービスの契約条件に従い、必要な保護措置を講じます。

16. セキュリティ

  • 外部サービスの秘密情報をアプリ内に保存しません。
  • 通信時は HTTPS / WSS / TLS を利用します。
  • 短命 token、署名、期限、利用量 lease により中継セッションを制御します。
  • サーバーログでは、音声本文、字幕本文、翻訳本文、認証情報、JWS全文を記録しないよう redaction を行います。
  • 端末内履歴および音声ファイルには、iOS / iPadOS のデータ保護機能を利用します。

17. 変更

当社は、必要に応じて本ポリシーを変更することがあります。重要な変更を行う場合、本アプリ内、Webサイト、App Store の更新情報その他適切な方法で通知します。第三者 AI サービスへの送信内容や送信先に重要な変更がある場合は、必要に応じてアプリ内で再同意を求めます。

18. 問い合わせ

本ポリシーまたはデータの取扱いに関する問い合わせは、サポートページよりご連絡ください。サポートに機密情報、個人情報、会議内容、音声データを送信しないでください。
