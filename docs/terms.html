BizCaption 利用規約

最終更新日: 2026-05-10  |  提供者: Kazuma Yoshimoto

重要: 本アプリの翻訳機能は、ユーザーの音声を Microsoft Azure OpenAI Realtime Translation に送信して処理します。利用前に、アプリ内のクラウド処理開示と プライバシーポリシー を確認してください。

1. 本規約への同意

本利用規約（以下「本規約」）は、Kazuma Yoshimoto（以下「当社」）が提供するリアルタイム音声翻訳アプリ「BizCaption」（以下「本アプリ」）の利用条件を定めます。

ユーザーは、本アプリを利用することで本規約に同意したものとみなされます。本規約に同意しない場合、本アプリの翻訳機能を利用しないでください。

2. サービス内容

本アプリは、ユーザーの音声または会議音声を文字起こし・翻訳し、翻訳テキストおよび翻訳音声を提供することで、会議、商談、通話、講演、学習等における理解と発話を補助するサービスです。

iOS / iPadOS 版では主にマイク入力による直接会話・字幕・翻訳・履歴確認を提供します。macOS 版では、BlackHole / Loopback 等の仮想オーディオデバイスと組み合わせ、会議アプリへ翻訳後の音声のみを出力したり、会議相手の音声を翻訳してローカルで聞いたりする会議モードを提供します。

3. 第三者 AI サービスの利用

本アプリのリアルタイム翻訳、文字起こし、翻訳音声生成は、Microsoft Azure OpenAI Realtime Translation に依存します。

ユーザーが翻訳セッションを開始すると、マイク音声または会議音声、翻訳元/翻訳先言語、セッション情報等が、当社の Edge Broker / Media Relay を経由して Microsoft Azure OpenAI Realtime Translation に送信されます。当社は、本アプリに Azure OpenAI API キーを保存せず、短命の認証情報と中継 token を用いて通信します。

外部クラウドサービス、Apple、通信回線、端末OS、会議アプリ、仮想オーディオデバイス、その他第三者サービスの障害、仕様変更、メンテナンス、提供停止、遅延、不具合により、本アプリの全部または一部を利用できない場合があります。

4. クラウド処理への同意

本アプリの翻訳機能を利用するには、アプリ内で表示されるクラウド処理開示に同意する必要があります。同意しない場合、または同意を撤回した場合、新規翻訳セッションは開始できません。

同意の対象、送信データ、送信先、利用目的、保存方針は プライバシーポリシー に定めます。重要な変更がある場合、当社は必要に応じて再同意を求めます。

5. 会議・録音・参加者同意に関するユーザーの責任

ユーザーは、本アプリを利用して会話・会議・商談・講演等の音声を取得し、Microsoft Azure OpenAI Realtime Translation に送信するにあたり、適用される法令、勤務先・取引先・会議主催者の規則、守秘義務、参加者への説明・同意取得義務を自ら確認し、必要な許可を得るものとします。

ユーザーがローカル音声保存を有効にする場合、録音、保存、再生、共有、削除についても、適用される法令、会議参加者への説明・同意、所属組織や取引先の規則を自ら確認し、必要な許可を得るものとします。

組織の規程、契約、秘密保持義務、顧客との合意により、クラウドサービスや第三者 AI サービスへの音声送信が禁止または制限されている場合、本アプリの翻訳機能を利用しないでください。

6. 翻訳精度と重要事項の確認

本アプリは、翻訳結果、文字起こし結果、翻訳音声、話者の意図、会議内容の完全性、正確性、即時性、低遅延性、特定目的への適合性を保証しません。

契約、価格、納期、仕様、法的判断、医療、金融、雇用、人命、安全に関わる判断など重要事項については、必ず原文、相手方、公式文書、専門家その他信頼できる手段で確認してください。

7. 仮想オーディオデバイスと会議アプリ

macOS 会議モードで Zoom、Microsoft Teams、Google Meet 等の会議アプリへ翻訳音声を出力するには、BlackHole、Loopback 等の仮想オーディオデバイスが必要になる場合があります。

会議アプリ、OS、仮想オーディオデバイスの設定を誤ると、翻訳音声が相手に届かない、元のマイク音声が混入する、音声がループする、会議相手の音声を取り込めない等の問題が発生する可能性があります。ユーザーは、会議前に音声ルートを確認し、必要に応じてテストを行ってください。

8. 端末内履歴・ローカル音声保存

本アプリは、翻訳履歴をユーザーの端末内に保存します。履歴には、会議・商談の内容、個人情報、営業秘密、顧客情報その他の機密情報が含まれる場合があります。

ユーザーがローカル音声保存を有効にした場合、音声ファイルも端末内に保存されます。ユーザーは、端末のロック、OS更新、端末紛失時の保護、履歴削除、保存済み音声削除など、自己の責任で適切な管理を行うものとします。

10. 無料利用枠

本アプリは、初回限定で累計2分まで翻訳機能を無料で利用できます。これは Apple StoreKit の無料トライアルではありません。

無料利用枠は、翻訳処理の利用量およびサーバー側記録に基づいて管理します。通信切断、バックグラウンド化、強制終了、不正利用の疑いがある場合、当社サーバー上の記録に基づき利用時間を補正することがあります。無料利用枠は一度きりであり、アプリの削除、再インストール、サブスクリプション解約、端末設定の変更によって再付与されません。
